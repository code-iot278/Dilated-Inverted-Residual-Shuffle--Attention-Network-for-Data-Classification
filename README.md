# Dilated-Inverted-Residual-Shuffle--Attention-Network-for-Data-Classification
QuantumInspired Feature Selection and Dilated Inverted Residual Shuffle-  Attention Network for Data Classification
 Dataset 1 (Adult Census Income): Contains 48,842 records with 14 demographic
and employment features. The target is binary income classification (≤50K / &gt;50K). It
provides a standard benchmark for feature selection and binary classification.
https://www.kaggle.com/datasets/wenruliu/adult-income-dataset
 Dataset 2 (Dry Bean Dataset): Contains 13,611 instances of 7 different dry bean
varieties with 16 geometric features (area, perimeter, etc.). The multiclass target
evaluates the framework’s ability to handle diverse, realworld measurement data.
https://www.kaggle.com/datasets/muratkokludataset/dry-bean-dataset
